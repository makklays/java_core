
Собственные свойства int[]

- .length — это единственное и самое важное поле (не метод!), которое возвращает размер массива. Обратите внимание: пишется без круглых скобок (в отличие от string.length()

Где же искать методы для работы с int[]?

Поскольку у самого массива методов нет, вся магия работы с ними скрыта в специальном классе-утилите java.util.Arrays. Это набор готовых статических методов.

Вот самые популярные из них:
- Arrays.toString(arr) — превращает массив в красивую строку для вывода на экран ([1, 2, 3]).
- Arrays.sort(arr) — быстро сортирует массив по возрастанию (изменяет сам исходный массив).
- Arrays.binarySearch(arr, key) — ищет элемент в отсортированном массиве за бинарное время (O(log n)). Возвращает индекс элемента.
- Arrays.copyOf(arr, newLength) — создает новый массив указанной длины и копирует туда данные из старого. Полезно для изменения размера массива.
- Arrays.copyOfRange(arr, from, to) — вырезает кусок массива (аналог substring у строк).
- Arrays.fill(arr, value) — заполняет весь массив одинаковыми значениями (например, можно сразу забить весь массив числами -1).
- Arrays.equals(arr1, arr2) — сравнивает два массива по их реальному содержимому (элемент за элементом).

Метода прямой сортировки по убыванию в классе Arrays действительно нет.
Однако писать свой алгоритм (например, пузырьковую сортировку) с нуля не нужно. На практике эту задачу решают тремя простыми путями.
Способ 1: Отсортировать по возрастанию и развернуть (Самый частый)
Способ 2: Использовать класс-обертку Integer[] и компаратор
Способ 3: Использовать Java Streams (Для Java 8 и выше)
