

1. Проверки и свойства строки

- length() — возвращает длину строки (количество символов).
- isEmpty() — возвращает true, если длина строки равна 0.
- isBlank() — возвращает true, если строка пустая или состоит только из пробелов (появился в Java 11).

2. Извлечение символов и подстрок

- charAt(int index) — возвращает символ, стоящий на указанном индексе (индексация с 0).
- substring(int beginIndex) — обрезает строку, начиная с указанного индекса и до конца.
- substring(int beginIndex, int endIndex) — возвращает подстроку с beginIndex (включительно) до endIndex (не включая его).

3. Преобразование регистра и очистка

- toLowerCase() — переводит всю строку в нижний регистр.
- toUpperCase() — переводит всю строку в верхний регистр.
- trim() — удаляет пробелы в начале и в конце строки.
- strip() — современный аналог trim(), который умеет удалять пробелы Юникода (появился в Java 11).

4. Сравнение строк

- equals(Object anObject) — проверяет две строки на полное совпадение текста (строки в Java всегда нужно сравнивать через equals, а не через ==).
- equalsIgnoreCase(String anotherString) — сравнивает строки, игнорируя регистр (например, "Apple" и "apple" вернут true).
- compareTo(String anotherString) — сравнивает строки лексикографически (по алфавиту). Возвращает 0, если строки равны, отрицательное число, если текущая строка «меньше» (идет раньше в алфавите), и положительное — если «больше».

5. Разделение, замена и объединение

- split(String regex) — разбивает строку на массив строк по заданному разделителю (например, по пробелу или запятой).
- replace(char oldChar, char newChar) — заменяет все старые символы на новые.
- replaceAll(String regex, String replacement) — заменяет все совпадения по регулярному выражению.
- join(CharSequence delimiter, CharSequence... elements) — статический метод, который объединяет несколько строк в одну через указанный разделитель.

6. Конвертация данных

- toCharArray() — превращает строку в массив символов char[] (очень часто используется в циклах for-each).
- String.valueOf(anyData) — статический метод, переводящий любой тип данных (число, массив, объект) в текстовую строку.
