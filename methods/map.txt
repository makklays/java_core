
1. Поиск и проверка

- containsKey(Object key) — проверяет, есть ли такой ключ в карте (возвращает true/false).
- containsValue(Object value) — проверяет, есть ли такое значение в карте (работает медленнее, чем поиск по ключу, так как перебирает всю карту).

2. Получение данных

- get(Object key) — возвращает значение по ключу. Если такого ключа нет, вернет null.
- getOrDefault(Object key, V defaultValue) — возвращает значение по ключу, но если ключа нет, вместо null вернет указанное вами значение по умолчанию. Очень полезен при подсчете количества элементов.

3. Добавление и изменение

- put(K key, V value) — добавляет пару «ключ-значение». Если такой ключ уже был, метод перезапишет старое значение новым и вернет старое значение.
- putIfAbsent(K key, V value) — добавит пару только в том случае, если такого ключа еще нет в карте (или если его значение равно null).

4. Удаление и очистка

- remove(Object key) — удаляет элемент по ключу и возвращает его значение.
- clear() — полностью очищает карту, удаляя все элементы.

5. Размеры и проверки состояния

- size() — возвращает количество пар «ключ-значение» в карте.
- isEmpty() — возвращает true, если карта пустая.

6. Обход карты (итерация)

- keySet() — возвращает множество (Set) всех ключей карты.
- values() — возвращает коллекцию (Collection) всех значений карты.
- entrySet() — возвращает множество пар (Set<Map.Entry<K, V>>), что позволяет в цикле одновременно читать и ключ, и значение.
