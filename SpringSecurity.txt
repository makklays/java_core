
Что такое Spring Security? В чем разница между авторизацией и аутентификацией?

Spring Security - часть Spring Framework представляющий механизмы для контроля за авторизацией пользователей. 
Для управления Spring Security необходимо создать конфигурационный класс-наследник WebSecurityAdapter.

Аутентификация - процедура проверки подлинности, например проверка подлинности пользователя путем сравнения введенного им пароля с паролем, сохраненным в базе данных

Авторизация - предоставление определенному лицу или группе лиц прав на выполнение определенных действий.
