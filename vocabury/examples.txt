
there... and it...

there will be - будет
there must be - должно быть / наверняка есть
there might be - может быть / возможно есть
there must have been - должно было быть / наверное было
there should have been - должно было быть (но не было или не факт)
there would have been - было бы (если бы…)
there is sure to be - точно будет / наверняка будет
there is bound to be - обязательно будет / неизбежно будет
there is going to be - будет (запланировано / ожидается)
there is likely to be - скорее всего будет
there is supposed to be - должно быть (по плану / ожиданию)
there used to be - раньше было

supposed - предполагаемый, считаемый 

Examples:
There will be a meeting — Будет встреча
There must be a mistake — Должна быть ошибка
There might be a problem — Возможно есть проблема
There must have been an error — Должно было быть ошибка
There should have been more people — Должно было быть больше людей
There would have been trouble — Были бы проблемы
There is sure to be a delay — Точно будет задержка
There is bound to be confusion — Неизбежно будет путаница
There is going to be a party — Будет вечеринка
There is likely to be rain — Скорее всего будет дождь
There is supposed to be a meeting — Должна быть встреча
There used to be a park — Раньше был парк

might < may < must (по степени уверенности)
